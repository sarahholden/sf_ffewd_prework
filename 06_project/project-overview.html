Front-End Bootcamp Prework Unit 2

About Me Page

Show off your HTML and CSS skills by creating a simple "about me" website.

The site should include a photo and some basic information about you.

Exercise

Requirements

  • an index.html file with HTML boilerplate and elements to structure the page (header, h1, p, and whatever makes sense for your site!)
  • a CSS file with styles applied by HTML tags, classes, or ids
  • in the HTML document’s head, a title and a link to your CSS file
  • at least one image

Here are the steps to set up the files and folders for your assignment:

  1. Create a new folder called on your desktop called yourname_about (with your own name).

  2. Open up the Atom text Editor.

  3. Go to File > Open... and select the yourname_about folder you created in the first step.

  4. Right click on the folder in the sidebar in Atom and select the New File option.

  5. Name this new file index.html.

  6. In this file, set up your HTML boilerplate code (and save the file).

  7. Add a title element so that your name shows in the browser tab.

  8. Create a file in folder named style.css.

  9. Now, go back to your index.html and connect your css file (style.css) by using a link tag to reference it in the head of your HTML file (make sure to save your HTML file after any changes):

    <link href="css/style.css" rel="stylesheet" type="text/css">

  10. You'll want to save each file after making any changes.

  11. To view your project in your browser, you can right click on your index.html file in the sidebar in Atom, select "Copy Full Path", and paste the path in the URL bar of your browser. Any time you make changes to your files and save those changes in Atom, you can refresh the page in your browser to see the updated results!

You should now be all set to start writing HTML and CSS!

Submission

  1. Make sure your index.html and your css file are linked together properly and that both are contained in a folder.
  2. Publish your site by uploading the folder to bitballoon (just drag and drop to the BitBalloon site!) Bitballoon will host your site on the web!
  3. Don't worry about getting this to "finished product" level - you're not an expert front-end developer yet!
